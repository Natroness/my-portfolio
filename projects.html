Syllable

Next.js, Google Gemini AI, ElevenLabs API, Tailwind CSS

Interactive PDF parsing tool with drag-and-drop upload, real-time progress visualization, and automatic date formatting for calendar integration. Integrated Google Gemini AI to convert academic documents into structured JSON data and added humorous audio summaries using ElevenLabs text-to-speech. Deployed on Vercel for fast performance and cross-platform accessibility.

Landmark AI

TensorFlow, Keras, iCrawler, Gradio

Deep learning image classifier (MobileNetV2) to recognize global landmarks from a custom-curated dataset. Automated dataset collection and preprocessing with iCrawler. Deployed a real-time web demo using Gradio for interactive landmark recognition.

Live Demo GitHub

Emoji Face Swap Application

Python, OpenCV, dlib, Flask, D3.js

Real-time facial landmark detection system that swaps faces with matching emoji expressions using OpenCV and dlib. Integrated into a Flask web app with interactive visualizations powered by D3.js.
